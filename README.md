### Git Requirements
1. Clone this project (one time only)
```
git clone https://github.com/lpeter29/E-Commerce-Website.git
```

2. Push your code
```
git add .
git commit -m "Message and Changes"
git push
```

3. Pull this Project
```
git fetch
git pull
```

4. Branching
```
git branch <branch_name>
git checkout <branch_name>
```
